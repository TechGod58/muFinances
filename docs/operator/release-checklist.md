# muFinances Release Checklist

## Before Promotion

- Release notes are drafted.
- Migration dry-run passes.
- Rollback plan is documented.
- Financial correctness fixtures pass.
- Playwright smoke tests pass.
- Performance benchmark is recorded.
- Backup and restore drill are current.
- Security readiness blockers are zero.

## Pilot Release

- Identify pilot users.
- Confirm pilot departments/entities.
- Enable monitoring.
- Confirm support path.
- Capture feedback and defects.

## Signoff

- Admin signoff.
- Controller signoff.
- Integration owner signoff.
- Security/IT signoff.
- Executive sponsor signoff where required.
