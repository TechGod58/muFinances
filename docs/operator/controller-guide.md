# muFinances Controller Guide

## Core Workflow

- Review budget submissions and approvals.
- Run close and reconciliation workflows.
- Validate import batches before posting.
- Review consolidation, eliminations, and audit packets.
- Approve report packages for release.

## Controls

- Posted or locked ledger lines are immutable.
- Import batches should move through staging, validation, approval, and posting.
- Reconciliation differences must be within tolerance or documented.
- Close tasks must complete before period lock.

## Review Areas

- Ledger audit chain.
- Reconciliation evidence.
- Variance explanations.
- SoD policy warnings.
- Board package validation results.
