# muFinances Planner Guide

## Core Workflow

1. Open the relevant workspace from the `Workspaces` menu.
2. Review assumptions, drivers, and department submissions.
3. Enter or import budget lines.
4. Add comments and attachments where evidence is required.
5. Submit the package for review.

## Planning Concepts

- Actuals are historical posted results.
- Budget is the approved planning baseline.
- Forecast is the current expected path.
- Scenario is a what-if planning version.

## Good Practice

- Use Excel templates for bulk edits when needed.
- Review validation warnings before submitting.
- Use variance comments for material changes.
- Do not use screenshots as evidence when source files are available.
