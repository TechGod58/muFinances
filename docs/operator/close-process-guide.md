# muFinances Close Process Guide

## Close Flow

1. Confirm all source imports are complete.
2. Review rejected import rows.
3. Complete account reconciliations.
4. Review aging and exception queues.
5. Post approved adjustments.
6. Run consolidation.
7. Generate audit packet.
8. Lock period after approval.

## Required Evidence

- Reconciliation support.
- Adjustment approval.
- Import validation/rejection reports.
- Consolidation audit report.
- Variance commentary.
- Close checklist completion.

## Lock Rule

After period lock, posted ledger lines are immutable. Corrections must be made through controlled adjustment workflows.
